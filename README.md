# html-demo-project

## create a new repository on the command line
1. echo "# html-demo-project" >> README.md
2. git init
3. git add README.md
4. git commit -m "first commit"
5. git branch -M main
6. git remote add origin https://github.com/shamayel-torabi/html-demo-project.git
7. git push -u origin main

## push an existing repository from the command line
1. git remote add origin https://github.com/shamayel-torabi/html-demo-project.git
2. git branch -M main
3. git push -u origin main
